# JavaScript Arrays and Functions

## Objective
A JavaScript program that receives an array as an argument and performs various operations using different types of functions.

## Tasks Completed
- Created an **Arrow Function** to find the maximum number in an array.
- Created an **Anonymous Function** to calculate the sum of all elements.
- Created a **Standard Function** to count the number of odd numbers.
- Tested all functions using the dummy array: `[4, 8, 2, 11, 6, 7, 10]`.

## How to View Output
1. Open `index.html` in a web browser.
2. Open the Developer Tools (F12 or Right-click > Inspect).
3. Go to the **Console** tab to view the results.
